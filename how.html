Overview #

LMSTFY Prank is a humorous web tool inspired by “Let Me Google That For You”. It automates typing a search query, simulates a click on the search button, and redirects to a search engine. The main purpose is entertainment and gentle humor, demonstrating automation on trivial searches.

Setup #

To set up LMSTFY Prank locally:

git clone https://github.com/GLTdotNS
cd lmstfy-prank
open index.html

Ensure JavaScript is enabled and modern browsers are used.

Features #

  • Automatic typing of queries
  • Auto click simulation
  • Redirect to multiple search engines
  • Lightweight, responsive design
  • Customizable for developers

Architecture #

The application is divided into three main modules:

  1. Input Handler: captures user input and triggers typing simulation.
  2. Typing Engine: simulates each character input with timing intervals.
  3. Redirect Controller: handles automatic click and redirection to the chosen engine.

Automatic Typing #

Typing simulation uses setInterval:

function typeInput(inputElement, query, callback){
  let i = 0;
  const interval = setInterval(()=>{
    inputElement.value += query[i++];
    if(i >= query.length){
      clearInterval(interval);
      callback();
    }
  }, 120);
}

Auto Click #

function autoClickButton(buttonElement){
  buttonElement.classList.add('clicked');
  setTimeout(()=>{
    buttonElement.classList.remove('clicked');
  }, 800);
}

Redirect Logic #

const engines = {
  google: q => `https://www.google.com/search?q=${encodeURIComponent(q)}`,
  bing: q => `https://www.bing.com/search?q=${encodeURIComponent(q)}`,
  duckduckgo: q => `https://duckduckgo.com/?q=${encodeURIComponent(q)}`,
  brave: q => `https://search.brave.com/search?q=${encodeURIComponent(q)}`
};

function redirectToEngine(engine, query){
  window.location.href = engines[engine](query);
}

Code Examples #

Combining typing and redirect:

const input = document.getElementById('query');
const button = document.getElementById('searchBtn');
const query = 'How to use LMSTFY';

typeInput(input, query, ()=>{
  autoClickButton(button);
  setTimeout(()=>redirectToEngine('google', query), 1000);
});

FAQ #

Q: Can I add more engines?
A: Yes, edit the engines object.

Q: Can I change typing speed?
A: Adjust interval in setInterval.
