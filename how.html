Overview

LMSTFY Prank is a web-based tool that simulates typing a search query and automatically redirects the user to a selected search engine. The purpose is entertainment, showing how a simple automation can humorously demonstrate searching for obvious questions. This documentation explains the architecture, JavaScript logic, and usage steps in detail.

Automatic Typing

The core feature is automatic typing, which simulates a user entering a search query character by character. This is achieved using setInterval:

function typeInput(inputElement, query, callback){
  let i = 0;
  const interval = setInterval(()=>{
    inputElement.value += query[i++];
    if(i >= query.length){
      clearInterval(interval);
      callback();
    }
  }, 120);
}

This function appends characters one at a time and calls a callback once typing is complete.

Auto Click

Once typing is finished, the button is automatically “clicked” for the prank effect:

function autoClickButton(buttonElement){
  buttonElement.classList.add('clicked');
  setTimeout(()=>{
    buttonElement.classList.remove('clicked');
  }, 800);
}

This provides a visual click effect and maintains the prank illusion.

FAQ

Q: Can I add more search engines?
A: Yes, add them to the engines object.

Q: Can I change the typing speed?
A: Adjust the interval in setInterval.

Q: Is this safe?
A: Yes, it only redirects to normal search engines for fun.
