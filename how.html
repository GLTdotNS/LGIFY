Overview

LMSTFY Prank is a humorous web tool inspired by "Let Me Google That For You". It automatically simulates typing a search query, clicks the search button, and redirects to the chosen search engine. The main purpose is entertainment and to gently tease people who ask obvious questions online.

Steps

  1. User enters a query on the main page.
  2. They select a search engine (Google, Bing, DuckDuckGo, Brave).
  3. The site simulates typing the query automatically.
  4. Once typing completes, the search button is automatically “clicked”.
  5. The user is redirected to the selected search engine with the query.

Examples

Here is a snippet showing the automatic typing and click logic:


function typeAndClick(query, button, callback){
  let i = 0;
  const interval = setInterval(()=>{
    input.value += query[i++];
    if(i >= query.length){
      clearInterval(interval);
      button.classList.add("clicked");
      setTimeout(()=>{ callback(); }, 800);
    }
  }, 120);
}

And redirecting to the search engine:


const url = "https://www.google.com/search?q=" + encodeURIComponent(query);
setTimeout(()=>{ location.href = url; }, 1200);

FAQ

Q: Can I customize the search engines?
A: Yes, you can modify the engines object in the JavaScript.

Q: Is this safe to use?
A: Absolutely, it only redirects to normal search engines and is purely for fun.
